ü¶ô Ollama Installation for Auditomatic Lite

This script configures Ollama to work with Auditomatic Lite by setting up the correct CORS permissions.

Quick Install

üçé macOS

Run this command in your Terminal:

curl -fsSL https://lite.auditomatic.org/ollama.sh | sh

ü™ü Windows

1. Download and install Ollama from ollama.com

2. Set the environment variable:

Variable: OLLAMA_ORIGINS
Value: https://*.auditomatic.org,http://localhost:3000

3. Restart Ollama

üêß Linux

1. Install Ollama:

curl -fsSL https://ollama.com/install.sh | sh

2. Configure CORS by editing the systemd service:

sudo systemctl edit ollama

3. Add these lines:

[Service]
Environment="OLLAMA_ORIGINS=https://*.auditomatic.org,http://localhost:3000"

4. Restart Ollama:

sudo systemctl restart ollama

What This Does

  • Installs Ollama (macOS only)
  • Configures CORS to allow connections from Auditomatic Lite
  • Makes the configuration persistent across reboots
  • Restarts Ollama to apply changes

Test Your Installation

After installation, verify Ollama is accessible:

curl http://localhost:11434/api/tags

Then visit Auditomatic Lite and enable Ollama in the settings!

Security Note: This configuration allows web pages from auditomatic.org domains to access your local Ollama instance. Only run this if you trust Auditomatic Lite.
